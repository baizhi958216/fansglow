# FanGlow / 应援棒

一个基于 Nuxt 4 + Tailwind CSS 的 PWA 应援棒网页应用，专为演唱会应援设计。

## ✨ 特性

- 🎨 **丰富的颜色选择** - 5种预设颜色 + 自定义颜色选择器
- 📝 **文字模式** - 支持中英日文，最多8个字符
- ✍️ **手写模式** - 在黑色画布上自由绘制
  - 支持图片上传和编辑（旋转、缩放、透明度调节）
  - 全屏编辑模式，内容自动适配
  - 预设应援棒图案（应援棒、爱心、星星、皇冠、音符、花朵、钻石）
- 💫 **5种动画效果** - 常亮、呼吸灯、慢闪、快闪、霓虹灯（彩虹）
- 📱 **全屏显示** - 支持全屏 API，点击退出
- 🔋 **防息屏** - 使用 Wake Lock API 保持屏幕常亮
- 📴 **离线可用** - PWA 支持，首次加载后完全离线
- 🚀 **快速启动** - 30秒内即可使用

## 🛠️ 技术栈

- **Nuxt 4.2.2** - Vue 3 全栈框架
- **Tailwind CSS** - 实用优先的 CSS 框架
- **@vite-pwa/nuxt** - PWA 支持
- **@vueuse/core** - Vue 组合式工具集
- **Canvas API** - 手写绘制功能


启动开发服务器 (http://localhost:3000):


构建生产版本:

```bash

```

预览生产构建:

```bash

```

## 📱 PWA 安装

1. 在支持的浏览器中打开应用
2. 点击浏览器菜单中的"添加到主屏幕"或"安装"
3. 应用将作为独立应用安装到设备上
4. 首次加载后即可离线使用

## 🎯 使用方法

1. **选择颜色** - 点击5个预设颜色之一，或使用自定义颜色选择器
2. **选择模式** - 文字或手写
3. **输入内容**:
   - 文字模式: 输入最多8个字符
   - 手写模式:
     - 在画布上自由绘制
     - 上传图片并调整（点击图片可旋转、缩放、调整透明度）
     - 使用预设图案（应援棒、爱心、星星等）
     - 点击全屏按钮进入全屏编辑模式
4. **选择动画** - 常亮/呼吸/慢闪/快闪/霓虹灯
5. **点击"立即应援"** - 进入全屏显示
6. **点击屏幕任意位置** - 退出全屏

## 🎨 颜色预设

- 初音绿: `#39C5BB`
- 红色: `#FF4444`
- 蓝色: `#4488FF`
- 粉色: `#FF88DD`
- 黄色: `#FFD700`
- 自定义: 使用颜色选择器选择任意颜色

## 🌟 动画模式

- **常亮** - 持续发光，无动画
- **呼吸灯** - 2秒周期的平滑渐变
- **慢闪** - 约1.5秒间隔闪烁
- **快闪** - 约0.5秒间隔闪烁
- **霓虹灯** - 彩虹色循环变化（五颜六色）

## 🖼️ 手写模式功能

- **自由绘制** - 使用手指或鼠标在画布上绘制
- **图片上传** - 支持上传图片到画布
- **图片编辑** - 点击图片后可以：
  - 拖动调整位置
  - 旋转图片（0-360度）
  - 缩放大小（0.1-3倍）
  - 调整透明度（0-100%）
- **预设图案** - 快速添加预设图案：
  - ✨ 应援棒 - 真实的应援棒造型
  - 💖 爱心
  - ⭐ 星星
  - 👑 皇冠
  - 🎵 音符
  - 🌸 花朵
  - 💎 钻石
- **全屏编辑** - 切换到全屏模式进行更大画布编辑
- **清空画布** - 一键清除所有内容
